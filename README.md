# 50TaskProjectManger
This is a simple project manager app

User Activities
Organization Admin
Highest role inside a tenant.
Activities:
•	Create and manage the organization workspace
•	Invite users to the organization
•	Assign roles to members
•	Manage organization settings
•	View all projects inside the tenant
•	Remove users
Project Manager
Manages projects within the tenant.
Activities:
•	Create projects
•	Assign team members
•	Create tasks
•	Monitor progress
•	Manage deadlines
•	Generate reports
Team Member
Execute tasks within assigned projects.
Activities:
•	View assigned tasks
•	Update task status
•	Upload files
•	Comment on task
•	Track work progress
Stakeholder / Client
Limited visibility role.
Activities:
•	View project progress
•	View reports
•	Provide feedback
•	Comment on deliverables

Onboarding Flow
Typical workflow when a new organization signs up.
1.	Organization registers.
2.	Chose a payment plan
3.	System creates a tenant record.
4.	First user becomes tenant owner.
5.	Owner invites team members.
6.	Members join the tenant’s workspace.
7.	Projects and tasks are created.
